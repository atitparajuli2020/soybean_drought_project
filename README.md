# soybean_drought_project
It provides the command line codes that was used for reads processing and variant calling of the deep sequenced lines used in the manuscript titled "Deep Short-read Sequences Facilitated Identification of Seven Putative Drought Tolerance Genes in a Genome-wide Association Study in Soybean" 

# reads processing and variant calling 
The detailed command line codes used in the analysis is presented as a bash file codes_used_manuscript. For replication of the result please use the version specified in the analysis to avoid running into any issues.
